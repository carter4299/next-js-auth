# Auth Starter in NextJS app router

## Usage

1. Download the inject.sh script and place it in your project root
```sh
curl -o inject.sh asjdkbj
# or 
# wget sahjbdhjkas -O filename.ext

```
```
.
├── inject.sh
├── public/
├── src/
├── node_modules/
```

---

2. Give exec privilege and run 
```sh
chmod +x inject.sh
./inject.sh
```
